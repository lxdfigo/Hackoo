Hackoo(颈椎病康复指南)是一款有趣的基于摄像头辅助操作的Chrome扩展

For install:
1、安装Chrome，解压Hackoo，将其添加至扩展中
2、安装Python2.7，以及一系列模块：PIB，pywin32，video capture...

For use:
1、打开浏览器，运行扩展，输入Yahoo ID密码login
2、启动本地python脚本camera.py，以及服务端脚本python -m CGIHTTPServer（由于暂时无服务器，就用localhost）
3、按照提示步骤操作，重新定位按F12，切换操控方向点击鼠标中键

ps:主功能以及完成，部分功能仍处于demo状态

have fun！
